# Changelog

## Version 2.1.0 - "Minimal" (Current)

### Design Overhaul
- **Complete redesign** with minimalistic, Vercel-inspired aesthetic
- Removed heavy gradients, animations, and decorative elements
- Clean typography with consistent font weights
- Simplified color palette (black, white, grey with royal blue accent)
- Increased whitespace and improved visual hierarchy

### Branding & Logo
- Standardized logo style across entire application (`text-2xl font-black`)
- Consistent "NEVER" (black) + "86" (royal blue #4169E1) branding
- Updated logos in:
  - All marketing pages (Landing, Features, Benefits)
  - Login page
  - Sidebar navigation
  - Public menu page
  - Footer components

### Marketing Pages
- **Landing Page**: Clean hero section, simplified stats, minimal cards
- **Features Page**: Tab-based role features with clean card design
- **Benefits Page**: Streamlined benefit cards with subtle borders
- All pages feature consistent navigation and footer
- Mobile-optimized with responsive typography and spacing

### Login Page
- Redesigned to match minimal home page aesthetic
- Removed animated backgrounds and decorative elements
- Simplified role selection (border-based, no gradients)
- Clean card design with minimal shadows
- Professional, client-facing appearance

### Navigation & Layout
- Fixed navigation bars with clean borders
- Consistent button styling (`h-9` for sm, `h-11` for lg)
- Simplified hover states and transitions
- Removed backdrop blur effects

### Technical Improvements
- Improved mobile responsiveness across all pages
- Consistent spacing and padding (`px-4 sm:px-6 lg:px-8`)
- Standardized card components with proper padding
- Clean, maintainable code structure
